## 依赖项

- Nator 电机控制 Dll
- Galaxy GxIAPINET.dll

### 配置大恒相机 DLL

右键项目依赖项 -> 添加项目引用 -> 浏览 -> 找到大恒安装目录下的 APIDLL\Win64\.NET6.0\GxIAPINET.dll -> 打上勾

重装 SDK 后可能会遇到相机传入的画面有条纹且很卡的问题，这时候可以试试卸载网卡后重新启动电脑，将网卡设置回到初始值，然后用SDK软件重新配置 IP，除此外不要再碰网络设置，这样就有可能能跑了。
